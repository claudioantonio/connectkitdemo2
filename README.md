Connectkit Demo using Next.js

How to run?

1. create a .env file in the root dir
```
APP_ALCHEMY_ID=
APP_WALLETCONNECT_PROJECT_ID=
```
2. run `run npm dev`


Check also the demo using [Vite](https://github.com/claudioantonio/connectkitdemo)
